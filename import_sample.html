2. Import Sample

After successfully importing the EasyProtoBuffer package, you can import the built-in sample project to quickly learn how to use the tool. The sample includes complete .proto files, serialization/deserialization examples, and a full workflow demonstration.

Step 1: Open the Sample Import Menu

In the Unity editor, go to the top menu bar and select Window > EasyProtoBuffer > Import Sample. A pop-up window will appear to confirm the import.

Step 2: Confirm Sample Import

In the pop-up window, review the sample content (including .proto files, C# demo scripts, and usage examples), then click the Import button to start importing the sample into your project.

Step 3: View the Sample Content

Once the import is complete, you can find the sample files in the "EasyProtoBuffer/Samples" folder in the Unity Project window. The sample includes:

  • Sample .proto files (demonstrating basic data structures)
  • Serialization/deserialization demo scripts
  • Example of how to use the generated C# code
  • Comments explaining each step of the workflow

What You Can Learn from the Sample

The sample project is designed to help you quickly get started with EasyProtoBuffer. You can learn:

  • How to write standard .proto files
  • How the tool automatically compiles .proto files into C# scripts
  • How to use the generated C# code for data serialization/deserialization
  • How to integrate the tool into your daily development workflow
// Sample .proto file included in the sample project
syntax = "proto3";

message PlayerData {
  int32 id = 1;
  string name = 2;
  int32 level = 3;
  float hp = 4;
}

You can run the sample scripts directly in Unity to see how the tool works in practice.

2.Global Protocol Buffers Version

EasyProtoBuffer supports all 3.x versions of Google Protocol Buffers. (21.0~34.1) ensuring compatibility with your existing .proto files and workflow.

Global version

Protoc and Runtime must be in the same version. You can confirm this in Easy ProtoBuffer/Config/version — the version number will be displayed in the Inspector panel.

Change version(optional)

You can change the version number to any one you like. Double-click the version file and select a text editor to modify it, then save the changes.

Important Notes

  • Ensure the version you select is from the official Google GitHub tag list
  • After changing the global version, when a .proto file is compiled, a download will be triggered. The tools and runtime will be synchronized to the new version, and the tool will automatically recompile all .proto files and update the corresponding C# code.
  • It is recommended to use a stable version (e.g., 34.1) for production projects to avoid potential runtime errors.

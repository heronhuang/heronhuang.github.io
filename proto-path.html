Target Path of a Single Proto File

By default, Easy ProtoBuffer compiles .proto files to the same path as the .proto file. However, you can customize the compilation target path for individual .proto files to fit your project’s folder structure and management needs.

Step 1: Select the Proto File

The Inspector panel will display the current target path,empty mean "./" .

Step 2: Change the Target path(Optional)

After setting a new folder name, click Apply — you will then see the new C# file located in the new location.

Note: Must delete the previous C# file compiled from the .proto file; otherwise, it will cause a duplicate definition

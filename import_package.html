1. Import Package

Import the EasyProtoBuffer package into your Unity project using the Unity Package Manager (UPM) to start using the tool. Follow the step-by-step guide below:

Step 1: Open Unity Package Manager

Launch your Unity project, then go to the top menu bar and select Window > Package Manager. This will open the Package Manager window.

Step 2: Add Package from Disk

Click the "+" button in the top left corner of the Package Manager window, then select Add package from disk from the drop-down menu.

Step 3: Select the Package File

Navigate to the folder where the EasyProtoBuffer package is stored, find and select the package.json file (the core file of the UPM package).

Step 4: Complete Import

Click "Open" to confirm. Unity will automatically import the package and all its dependencies. Once the import is complete, the tool will be integrated into your project's asset pipeline, and you can start using its features.

// Note: Ensure the package.json file contains the correct dependencies
// Example package.json structure (simplified):
{
  "name": "com.yourcompany.easyprotobuffer",
  "version": "1.0.0",
  "displayName": "EasyProtoBuffer",
  "description": "Automatic protobuf compilation tool for Unity"
}

After importing, you can verify the package is installed by checking the "Installed Packages" list in the Package Manager.
